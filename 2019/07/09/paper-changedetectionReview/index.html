本篇论文是由印度的Shivangi Mishra, Priyanka Shrivastava and Priyanka Dhurvey所作的Change Detection Techniques in Remote Sensing —— A Review。

Abstract

地球表面由于自然现象或人类活动而不断变化。识别随时间在地球表面发生的变化的过程称为变化检测。利用多种技术在遥感领域有效地进行了地表变化探测。本文的目的是比较分析地表变化的各种变化检测技术

Introduction

地球表面的变化是由于自然灾害、森林砍伐、河流变粗、城市化等原因造成的。地球表面变化可分为土地利用和土地覆盖两大类。土地用途一词是指特定的一块土地用于农业、城市化、采矿等目的。土地覆盖(land cover)一词代表地表的特征，如建筑物、人行道、树木等。
如果能够及时准确地对地球表面的变化进行检测，那么就可以更好地分析和理解自然现象与人类之间的关系和相互作用，从而更好地管理和利用资源。
变化检测涉及到利用多时间数据集对土地覆盖类变化进行定量分析。
变化检测可以采用传统方法和遥感技术进行。传统的方法成本高、耗时长、精度不高，而遥感技术不能很好地解决这些问题。

Remote sensing technology

遥感被定义为一门科学和艺术，它通过分析一个设备所获得的数据来获得关于一个物体、区域或现象的信息，而该设备与所研究的物体、区域或现象并没有接触。
遥感分为主动遥感和被动遥感两种类型。主动遥感发射自己的电磁辐射，与物体相互作用或雷达和激光雷达等区域。被动式遥感系统利用太阳辐射照射被观测的物体或区域。从表面或物体反射或发射回来的能量由传感器记录下来。这些传感器可以是空基的，也可以是空基的。遥感系统获取的数据以航空照片、卫星图像、空间数据集等数据形式存在。在遥感的帮助下，有可能收集到有关这一地区的数据。由于数据量大，遥感已在灾害评估、海岸应用、农业应用、自然资源管理等方面得到了广泛的应用。
因为传感器记录的所有辐射都必须穿过大气层。因此，大气成分影响辐射，并引起辐射的吸收和散射。因此，数据质量下降。大气的一些影响可以从遥感系统中获得，通常是数字图像的形式。为了从图像中提取有用的信息，需要对得到的图像进行图像处理，从而增强图像，有助于更好的视觉解释。如果图像受到几何畸变、模糊或其他因素的影响，图像处理还可以纠正和恢复图像。

Change detection techniques

为了探测和分析地球表面的变化，采用了各种技术。在研究各种变化检测技术之前，有必要了解变化检测的过程。为了探测地表的变化，Jensen提出了六个重要的步骤:

  1. 变更检测问题的性质。
  2. 选择遥感数据。
  3. 图像预处理。
  4. 图像处理或分类。
  5. 变更检测算法的选择。
  6. 变更检测结果的评估。

变化检测的目的是识别数字图像上描述两个或多个图像日期之间感兴趣的特征变化的区域。变化检测过程的可靠性可能受到各种环境因素的强烈影响，这些环境因素可能在图像日期之间发生变化。常用的变化检测技术有:

Post classification

这种方法也称为delta分类。后分类方法被证明是变化检测分析中最常用的方法。需要对独立生成的分类图像进行比较。该方法首先对分类后的图像进行独立的校正，生成专题图，然后对比相应的标签，识别出发生变化的区域。
如果对时间为T1和T2的分类结果进行适当编码，就可以很容易地得到显示完整变化矩阵的变化图。
这一技术有几个优点，即最大限度地减少传感器、大气和环境的差异，这是由于来自两个日期的数据是分开分类的，从而最大限度地减少了两个日期之间的大气和传感器差异的归一化问题.

Image ratioing

对于识别变化区域，图像比值法被认为是一种快速的方法。通过计算两个注册图像在不同日期、一个或多个频带上对应像素的DN值的比值来进行图像比率。数据之间的比较是逐像素进行的。
在变动方面，根据这两个日期之间变动的性质，比率将大于或小于1。如果Rkij的值等于1，则表示没有变化。
该技术的问题在于，在更改和不更改之间选择阈值是一项具有挑战性的任务。由于合成图像直方图的非正态分布，对该方法提出了批评。

Image differencing

该方法采用逐像素、逐频带的方法，对不同时间采集的两幅空间注册图像的DN值进行减除。
如果DN值之间的差值为0，则表示没有变化。如果发生了变化，值要么是正的，要么是负的。该技术的问题是识别阈值的变化和没有变化的结果图像。标准差作为阈值选择的一种手段。
在图像差分方法中，即使地球表面没有发生变化，有时也会出现差异，这是由于在不同的日期获得的图像从未获得准确的配准和完美的辐射测量条件。

Principal component analysis

该方法基于方差的特征向量或合并数据集的协方差矩阵。
该方法是一种多变量分析技术，它将原始多光谱图像中的光谱分量减少到较少的主成分，而主成分是原始多光谱图像中出现大部分方差的原因。对于谱分量的约简，通常采用线性变换。对多日期图像进行线性变换，并进行组合，得到单个数据集。在进行多时间研究时，对两个或多个日期的主成分进行比较。
这是一种依赖于场景的技术，因此，很难解释发生在该地区。

Direct multidate classification

该方法将多数据图像组合成一个数据集，然后对组合后的图像进行分类。
直接多数据分类的方法可以是监督的，也可以是非监督的。在监督模式下，针对变化和无变化区域确定训练集。在无监督方法中，只检查场景的一小部分，以便发现发生更改的地方。在这两种情况下，变更类的统计数据都不同于非变更类。
这种方法产生了许多类，这些类显示了单幅图像中的光谱变化&也显示了在各种毁损图像中发生的时间变化。在这种方法中，结果的解释是一个相当复杂的任务&它还需要对感兴趣的领域有很好的知识。

Change vector analysis

在这种方法中，所发生的变化是用光谱空间中的空间表示的。当这种变化随着时间的推移而发生时，预计像素会改变其在光谱空间中的位置。这个位置的变化用一个向量表示。这个向量叫做光谱变化向量。
光谱变化矢量不仅定义了变化的方向，而且还定义了变化的大小。方向提供了变化的性质，大小提供了关于已经发生的变化水平的信息。
此方法提供有关所发生更改的详细信息。与此方法相关的问题是，很难定义量值阈值，而量值阈值可以区分变化和不变化，并且可以解释具有变化性质的向量的方向。

Change detection procedure

图像选择->图象配准->辐射和大气校正->时段遥测分析->变化检测

Selection of change detection technique

很难确定最适合于确定在感兴趣的领域中发生的变化的方法。在选择最适合的变化检测技术时存在一定的问题，因为各种特征的物理特征随时间而变化，图像配准不完善，云/雾的影响，不同技术的优缺点。
选择最适合的变化检测技术是一项相当艰巨的任务，因为单一的方法不能适用于所有的问题，问题的性质决定了哪种技术最适合使用。在选择变更检测技术时，需要考虑各种约束条件，如所需信息、数据质量、应用领域等。
通过确定变化检测研究的对象，可以选择合适的变化检测技术。只有在需要变化和无变化信息的情况下，才能使用图像差分和图像比值等技术。如果需要一个详细的矩阵，就必须采用员额分类等技术。研究区域的大小和空间分辨率对特定变化检测技术的选择起着至关重要的作用。
要成功地实现特定的变更检测技术，各种因素起着关键作用。这些因素包括不同时间获取的图像之间的配准质量、大气条件、采集时间、光照、视角、土壤湿度、噪声、图像中的阴影、植被的变化、传感器的校准、感兴趣区域的地形、分析人员的技能和经验。技术实现中涉及的不同步骤会导致各种错误，进而影响变更检测的成功实现

Conclusion

多日期图像中DN值的差异可以作为该区域发生变化的指标。多日期图像中DN值的变化可能是由于地球表面的变化、光照差异、大气条件、传感器标定、多日期图像配准不当等原因造成的。变化检测技术的成功实现依赖于两幅图像配准的准确性。
利用遥感变化检测技术可以绘制出土地利用和土地覆被随时间变化的地图。地球上的这些变化是由人类活动和灾难造成的。
变化检测已经得到了它的各种应用，其中很少有如下几点:

  1. 森林砍伐。
  2. 作物监测。
  3. 土壤含水量。
  4. 城市规划。
  5. 水质等。

收获

本文主要介绍了遥感图像中变化检测的步骤。主要有以下几步：
图像选择->图象配准->辐射和大气校正->时段遥测分析->变化检测。

变化检测通常有这几种方法：

  1. Post classification
  2. Image ratioing
  3. Image differencing
  4. Principal component analysis
  5. Direct multidate classification
  6. Change vector analysis
