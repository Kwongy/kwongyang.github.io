括号

小括号和中括号可以直接使用 () []
大括号需要用反斜杠转义 \{ \}
在Mathjax中大括号是用来分组的直接使用大括号表示分组x^{10}表示$x^{10}$
括号的伸缩使用\left和\right，\left(\frac{\sqrt x}{y^3}\right)表示$\left(\frac{\sqrt x}{y^3}\right)$
绝对值符号\vert x \vert，表示为$\vert x \vert$
范数符号\Vert x \Vert，表示为$\Vert x \Vert$
尖括号\langle x \rangle，表示为$\langle x \rangle$
向上取整符号\lceil x \rceil,表示为$\lceil x \rceil$
向下取整符号\lfloor x \rfloor，表示为$\lfloor \rfloor$

求和、极限与积分

  • 求和\sum_{i=1}^n{a_i}，表示为$\sum_{i=1}^n{a_i}$
  • 极限\lim_{x\to 0}，表示为$\lim_{x\to 0}$
  • 积分\int_0^\infty{fxdx}，表示为$\int_0^\infty{fxdx}$

向量、重音符和变音符

  • 向量\vec x，表示$\vec x$。\overrightarrow {xy}，表示$\overrightarrow {xy}$
  • 横线\bar x，表示$\bar x$。\overline {xyz}，表示$\overline {xyz}$
  • 重音符\hat x，表示$\hat x$
  • 变音符\widehat {xy}，表示$\widehat {xy}$

分式与根式

  • 分式\frac{xy}{z}，表示为$\frac{xy}{z}$
  • 根式\sqrt[x]{y}，表示$\sqrt[x]{y}$

空格

Mathjax在渲染时会忽略空格。
小空格a\ b，表示为$a\ b$
4个空格(Tab键)a\quad b，表示为$a\quad b$

特殊符号

样式 写法

特殊函数

直接使用\函数名即可例如

  • \ln，表示为$\ln$
  • \max，表示为$\max$
  • \sin，表示为$\sin$
